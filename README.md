# ReStaff Avatars — v11 Simple Auth

## 🔐 Вход — email домен + общий пароль

Никакого Resend, никаких DNS, никаких писем. Простой механизм:

1. Юзер вводит email — мы проверяем что домен в списке разрешённых:
   - `@restaff.pro`
   - `@restaff.tech`
   - `@staffco.ru`
2. Юзер вводит общий пароль команды: **`Успех неизбежен`**
3. Если ОК — сессия на 30 дней в localStorage

> 💡 Пароль захардкожен в `app.js` (константа `TEAM_PASSWORD`). Это нормально для внутреннего сервиса — реальной защиты пароль в JS не даёт (любой может его открыть в DevTools), но отсекает случайных посетителей. Главная защита — что сервис доступен только на корп.домене и о нём знают только сотрудники.

### Сменить пароль
Открой `app.js`, найди строку:
```js
const TEAM_PASSWORD = 'Успех неизбежен';
```
Поменяй значение, закоммить, Vercel передеплоит. После смены — всех текущих юзеров **не разлогинит** (у них уже есть сессия на 30 дней). Полностью обновить можно почистив localStorage у всех (или подождать 30 дней).

## 🚀 Деплой

Залить в GitHub репо `restaff-avatars` **2 файла**:
1. `index.html` — заменить
2. `app.js` — заменить

Файлы `vercel.json`, `assets-data.json`, `api/remove-bg.js` не меняются.

Vercel автоматически передеплоит через ~30 сек.

**Если ранее заливал файлы `api/auth-send-code.js` и `api/auth-verify-code.js`** — удали их из репо (через GitHub UI: открой файл → меню троеточие справа → Delete file). Они больше не нужны.

В Vercel env переменные `RESEND_API_KEY`, `AUTH_SECRET`, `AUTH_SENDER_EMAIL` можно тоже удалить (но это не обязательно — лежат и не мешают).

- Кнопка **«Убрать фон» теперь ВЫШЕ кадрирования** — порядок шагов логичен (сначала убрать фон → потом подогнать кадр)

## ✅ Чеклист после деплоя
1. Открой restaff-avatars.vercel.app
2. **Cmd+Shift+R** / **Ctrl+Shift+F5** — жёсткая перезагрузка чтоб подхватить новый JS
3. Жми «Войти»
4. Email: твой рабочий → Пароль: `Успех неизбежен`
5. Должен залогинить и попасть на «Шаг 1 · Подготовь фото»
6. Прогони весь флоу: фото → убрать фон → подогнать → стилизация → скачать

Можно звать команду тестить!
