# ReStaff Avatars — v10 Final

## 🔐 Новое: вход по 6-значному коду на email

Демо-вход заменён на полноценный с подтверждением:
1. Юзер вводит email
2. На email приходит 6-значный код (действует 10 минут)
3. Вводит код → сессия на 30 дней

Левый посетитель не зайдёт даже зная домен — нужен доступ к рабочей почте.

### Настройка Resend в Vercel (5 минут, 1 раз)

#### Шаг 1: API ключ Resend
- Регистрация на https://resend.com → Sign Up
- **API Keys** → **Create API Key** → "ReStaff Avatars", scope **Sending access**
- Скопировать `re_...`

#### Шаг 2: Верифицировать домен
- В Resend → **Domains** → **Add Domain** → `restaff.pro`
- Resend покажет DNS записи (SPF, DKIM, DMARC) — добавить их в DNS вашего домена
- Дождаться статуса **Verified** (5-30 минут)

#### Шаг 3: Env переменные в Vercel
Settings → **Environment Variables**:

| Имя | Значение |
|---|---|
| `RESEND_API_KEY` | `re_xxx...` из шага 1 |
| `AUTH_SECRET` | случайные 32+ символов (см. ниже) |
| `AUTH_SENDER_EMAIL` | `noreply@restaff.pro` (любой адрес на верифицированном домене) |

Генерация `AUTH_SECRET`:
```bash
node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"
```

Vercel автоматически передеплоится после добавления переменных.
