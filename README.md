# ReStaff Avatars — Release v2 (Torso style)

Корпоративный сервис создания аватарок с новым стилем «торс выходит за круг».

## Как обновить прод на Vercel

### Вариант 1: Загрузить новые файлы поверх старых

В репозитории `restaff-avatars` на GitHub:

1. **Add file → Upload files**
2. Перетащи **`index.html`** и **`app.js`** (заменят старые версии)
3. **assets-data.json** — оставь как есть, в нём логотип
4. **api/remove-bg.js** — уже есть, не трогаем
5. Внизу — **Commit changes**

Vercel автоматом пересоберёт через 30 секунд.

### Вариант 2: Полная замена

1. Удалить старые `index.html` и `app.js` через GitHub UI
2. Загрузить новые из этой папки
3. Commit

## Что нового

### Визуальная концепция
- **Стиль "Torso"**: голова сверху круга, плечи выходят за нижнюю границу
- **Цветной круг подложка** с градиентом цвета подразделения
- **Неоновое свечение** вокруг круга (настраивается)
- **Лого RE на футболке** опционально (имитация мерча)

### Цвета подразделений
| Отдел | Цвет |
|---|---|
| Топ менеджмент | Премиум-чёрный + золотой блик + настраиваемый неон |
| Продажи | Красно-оранжевый |
| Маркетинг | Бриллиантовый зелёный |
| Финансы | Янтарно-жёлтый |
| Юристы | Циан |
| Разработка | Насыщенный синий |
| Поддержка | Фиолетовый |
| Аналитика | Изумруд → мята |
| HR | Коралловый |

### Регион
- **РФ**: белый фон
- **Global**: тёмный фон

### Кринж-режим (для команды)
Открывается снизу страницы:
- 💉 Тату «Успех неизбежен» (текст фиксирован) — позиционируется
- 🐱🐶 Питомец сбоку — заглядывает в аватарку
- 🪪 Селфи с паспортом
- ⚠️ «Выкатываем СБП на прод» — красный фильтр, молнии, тёмные оверлеи
- 🥛 Творожник — выгорание, серый, таблетки, кофе, черепа
- 🇧🇾 Беларусь — флаг + гора картошки
- 🎤 Караоке (Грузия) — флаг + микрофон + вино
- 🚩 Красный флаг
- 🏖️ Отпуск — фото пропадает, в круге граффити «Отпуск»
- 🎉 МАКС КРИНЖ — открытка из ватсапа

## Технические детали

- **Удаление фона**: через Vercel Serverless Function `/api/remove-bg`, проксирует remove.bg API. Ключ в env переменной `REMOVE_BG_API_KEY`.
- **Размер круга**: фиксированный 32% от ширины канваса
- **Композиция**: круг ровно по центру холста, голова естественно выходит за верхнюю границу
- **Авто-фильтры**: фильтры применяются ТОЛЬКО к лицу через `globalCompositeOperation`
- **Шрифты**: Inter, Caveat (тату/стихи), Permanent Marker (граффити), Lobster, Monoton, Bungee Shade
